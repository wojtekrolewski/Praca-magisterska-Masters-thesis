# Praca-magisterska
# Porównanie wybranych algorytmów uczenia struktury sieci bayesowskich

## 1. Problem
Celem pracy było porównanie wybranych algorytmów uczenia struktury sieci bayesowskich
pod względem jakości uzyskanych modeli oraz złożoności obliczeniowej.

## 2. Algorytmy
- algorytmy oparte na przeszukiwaniu
- algorytmy score-based
(opis intuicyjny, nie matematyczny)

## 3. Dane
- dane syntetyczne
- zestawy testowe o różnej liczbie zmiennych

## 4. Metodologia
- kryteria oceny jakości struktury
- porównanie czasu obliczeń
- powtarzalność eksperymentów

## 5. Wyniki
- zestawienie jakości modeli
- porównanie złożoności obliczeniowej

## 6. Wnioski
- kiedy dany algorytm ma sens
- kompromis jakość vs. czas
- ograniczenia podejść

## 7. Technologie
- R
- pakiety do analizy sieci bayesowskich (bnlearn)
